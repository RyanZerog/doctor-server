# 面瘫检测及评估系统 - 医生端

面瘫检测及评估应用的医生用Web端管理系统，用于医生管理面瘫患者信息、分类及随访提醒。

## 功能特点

- 患者标签分类管理：医生可以创建、编辑标签，并将患者归类到不同标签下
- 面瘫原因分类：支持按照面瘫原因（先天性、BELL面瘫、外伤、肿瘤、医源性、不明原因/其他）进行患者分类
- 随访提醒功能：系统默认设置三个月复查提醒，支持自定义随访周期
- 患者详情管理：记录患者基本信息、面瘫情况、随访计划等
- 现代化界面：符合医疗领域专业审美，操作便捷

## 技术栈

- React 18
- React Router v6
- Ant Design 5.x
- Zustand (状态管理)
- Dayjs (日期处理)

## 开发环境启动

确保已安装Node.js环境（建议使用v16+），然后执行：

```bash
# 安装依赖
npm install

# 启动开发服务器
npm start
```

## 构建生产版本

```bash
npm run build
```

构建后的文件将位于 `build` 文件夹中，可部署到任何静态文件服务器。
